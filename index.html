Python Calculator

Developed a program in Python that accepted multiple polynomials of any degree as a user-input and performed mathematical operations; addition, subtraction, multiplication, division, derivation and integration. Implemented Object Oriented Programming through classes to represent polynomials as objects with specific traits to optimize the program. Created methods for each of the mathematical operations that accepted specific parameters and returned the resulting polynomial after the operation was evaluated.

Y-Hacks 2019: Biased

Created in a group of 4 a Google-Cloud-Vision enabled web-scraping mobile platform that recognizes logos and returns paragraphs that are strongly related to revealing the social-biases of a company. My main role within this project was to develop a web-scraping algorithm in JavaScript to get information from Wikipedia relating to the company that had certain key words we were interested in receiving information on such has climate change, environment, pollution etc. Was also in charge of hosting the web application on firebase as a PWA (Progressive Web Application) for increased efficiency and overall better user experience.

Bluetooth-Remote Control Arduino Car

Used laser printing to get the required components for the base structure of the vehicle. Secondly, used wires and soldering to connect the motors and the circuits. Programmed the Arduino using python library AFMotor.h which can detect user inputs from an application that can be connected to via Bluetooth from a phone.
